## Tagger - Vanilla JavaScript Tag Editor

[![npm](https://img.shields.io/badge/npm-0.1.1-blue.svg)](https://www.npmjs.com/package/@jcubic/tagger)

Usage:

```
tagger(document.querySelector('[name="tags"]'), {allow_spaces: false});
```
